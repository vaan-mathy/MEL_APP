Help - Software Usage and Installation

Overview

This software is a web-based student management system that allows you to add, edit, delete, and view student records with training history. It is built with Node.js backend and uses MySQL as the database.

How to Use

  • Login Page: Use your credentials to log in.
  • Home Page: Navigate to Student E-Data to manage student information or Instructors for instructor data.
  • Student E-Data Page: Search and filter student records using multiple filters.
  • Student Detail Page: View full student details, edit information, add or remove training records, save changes, or delete the student.
  • Add New Student Page: Input new student information and save it to the system.
  • Help Button: Click the help button on any page for guidance.

Installing Dependencies and Running the Software

To install and run this software locally, follow these steps:

  1. Install Node.js:
    • Download and install from the official website: https://nodejs.org/
    • Verify installation by running node -v and npm -v in your terminal or command prompt.
  2. Install MySQL:
    • Download and install from the MySQL official website: https://dev.mysql.com/downloads/mysql/
    • Create a database for the application and configure user credentials.
  3. Clone or Download the Software Code
  4. Install Project Dependencies:

    Navigate to the project directory in your terminal and run:

    npm install
  5. Configure Database Connection:

    Update the database connection settings in your Node.js application's configuration file (e.g., config.js or environment variables) to connect to your MySQL database.

  6. Initialize the Database:

    Run the provided SQL scripts or migrations to create required tables and seed initial data.

  7. Run the Application:

    Start the Node.js server:

    npm start

    Or if configured differently:

    node app.js
  8. Access the Application:

    Open your web browser and go to http://localhost:3000 or the configured port.
